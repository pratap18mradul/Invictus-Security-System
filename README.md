# 🛡️ Invictus Security System

A Progressive Web App (PWA) that helps users send emergency SOS alerts with live GPS location through SMS and WhatsApp.

## 🚀 Live Demo

https://pratap18mradul.github.io/Invictus-Security-System/

## ✨ Features

- 📍 Live GPS Location
- 🆘 Emergency SOS Button
- 🎙️ Voice Assistant
- 📱 WhatsApp Integration
- 💬 SMS Support
- 👥 Emergency Contacts
- 📜 Activity History
- 🌐 Offline Support (PWA)

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Geolocation API
- Web Speech API
- Local Storage
- Service Workers
- Progressive Web App (PWA)
